Cloudinary image loader demo

What does this loader do?

  • Responsive sizing + lazy loading
  • It can load images with a maximum width
  • It can load images with a fixed width
  • It uses clean data-src with your server url
  • It uses customizable Cloudinary images presets
    like w_auto,f_auto,dpr_auto,c_scale
